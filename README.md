# R Data Mining and Machine Learning Assignments

This repository contains a structured set of academic assignments implemented using the R programming language. It demonstrates core concepts of Data Mining, Machine Learning, Statistical Analysis, and Predictive Modeling.

---

## 📌 Topics Covered

- Data Exploration & Statistical Analysis
- Descriptive Statistics
- Hypothesis Testing (Z-Test & T-Test)
- Association Rule Mining (Apriori Algorithm)
- Linear Regression
- Logistic Regression & ROC Curve
- Naive Bayes Classification
- Decision Tree Classification
- K-Means Clustering
- Random Forest Classification

---

## 🧠 Algorithms Implemented

- Apriori Algorithm (Market Basket Analysis)
